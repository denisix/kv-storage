# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-14

### Added
- Initial release of kv-storage-client
- HTTP/2 client for KV Storage server
- Bearer token authentication
- Async/await API using tokio
- PUT, GET, DELETE, HEAD operations
- Batch operations support
- Paginated key listing
- Health check and metrics endpoints
- Automatic key percent-encoding for safe URI transport
- Connection pooling and session management
- Configurable timeouts and concurrent streams
- Comprehensive error handling
- Full documentation and examples
